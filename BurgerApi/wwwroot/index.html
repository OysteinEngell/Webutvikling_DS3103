Burger API Documentation

This page provides documentation for the Burger API endpoints.

Go back to application

Getting started

  • Make sure that you have installed the .NET SDK.
  • To run the API use the command: dotnet watch run
  • If the port provided when running the server is different from the provided endpoint,
    you will need to update the endpoint inside the BurgerApiService and UploadImageService module located in the src folder within the frontend directory.

Endpoints

Endpoints are added on to the base url: "https://localhost:7130"

NOTE: the port may be different when running the server

Endpoint Method Description Parameters Response
/burger GET Get all burgers N/A List of Burger objects
/burger/{id} GET Get a specific burger by ID id: integer Burger object
/burger/search/{name} GET Search for burgers by name name: string List of Burger objects
/burger POST Create a new burger Burger object (JSON payload) Newly created Burger object
/burger/burgerList POST Create multiple burgers List of Burger objects (JSON payload) HTTP Status 200 if successful
/burger PUT Update an existing burger Burger object (JSON payload) HTTP Status 200 if successful
/burger/{id} DELETE Delete a specific burger by ID id: integer HTTP Status 200 if successful
